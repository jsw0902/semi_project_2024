JQuery

기존에 자바스크립트 언어로 구현하기에 복잡하고 길었던 구문들을
간편하게 사용할 수 있도록 단순화시킨 Javascript 기반의 라이브러리.

Jquery 사용 시, 이점
1) DOM 요소와 관련된 스크립트를 보다 간결하게 구현할 수 있다.
2) Ajax(비통기 통신), 이벤트 처리 등 다양하게 지원함.

jQuery 라이브러리 연결 방법

1. 온라인 방식 : CDN(Content Delivery Network)
- head 태그내에, script 태그를 작성하여 jquery가 작성되어있는 url과 연결하여 사용하는 방식
- 인터넷이 연결되어 있는 상태에서 사용 가능

2. 오프라인 방식 : Jquery 라이브러리를 다운로드 받아서 사용하는 방식 - Jquery 파일을 다운로드 받아, 적절한 폴더에 위치시키고, script 태그를 작성하여 jquery 파일이 위치한 경로를 지정하여 사용하는 방식
